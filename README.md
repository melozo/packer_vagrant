##  POC

* Tres entornos funcionales que puedan ejecutarse en vagrant
* 1 nginx
* 2 php, ¿mysql?
* 3 redis
* test: https://github.com/Tombar/sample-app 


## Uso
```bash
git clone blaublau && cd blaublau
./make.sh #show help
./make.sh php
```
